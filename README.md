# httproxy
A simple HTTP/HTTPS proxy

# Installation

`go get -u github.com/justmao945/httproxy`

# Usage

* Install httproxy on the proxy machine and start it with `httproxy -addr=:1081`.
* Can work with [KCPTUN][2] to build a tunnel http proxy...
